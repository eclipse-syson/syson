* In diagrams:

** Improve diagram-to-diagram drag and drop to support dropping multiple graphical nodes at once.

A _frame_ graphical edge can be created between a `RequirementUsage` or a `RequirementDefinition` graphical node and a `ConcernUsage` graphical node.
